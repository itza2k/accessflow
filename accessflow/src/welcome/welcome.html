Welcome to AccessFlow

AI-powered simplification and accessibility for web content

Step 1: Set up your API key

AccessFlow uses Google's Gemini API to process text. You'll need to provide your own API key to use this extension.

Don't have an API key? Get a free API key from Google AI Studio

Step 3: Get Started

Using AccessFlow is easy:

  1. Select text on any webpage
  2. Right-click and choose "Process with AccessFlow" or use the keyboard shortcut Ctrl+Shift+A
  3. Choose to simplify or summarize the content
  4. Explore key concepts and use read aloud features
